DESCRIPTION DU SITE - ASSISTANT TOURNÉES (Pour les nulls 🤪)

🎯 QU'EST-CE QUE C'EST ?

Assistant Tournées = Application web PWA (Progressive Web App) pour La Poste - gestion ultra-simple des tournées de distribution.

Logo : Sniper (assets/snipers.png) → "Précision chirurgicale" pour trouver les adresses ! 🥷

Thème : Bleu La Poste + dark mode automatique + PWA installable (manifest.json)

🏠 ACCUEIL (Mode par défaut)

1. Sélection BRAS (obligatoire)

[Boutons bleus : Bras 1, Bras 2, Bras 3...]

→ Cliquez un BRAS → Active (bleu) → Recherche activée !

2. VILLES (optionnel - apparaît après BRAS)

[Paris, Lyon, Marseille...]

→ Filtre fin (clic toggle ON/OFF)

3. RECHERCHE (zone principale)

  • Micro → Dites "RUE DE LA PAIX" → Popup confirme "PAIX ?" → OK
  • Type 1 (exact) → Type 2 (approx si vide)

→ Rue de la Paix     CS    ← ROUGE (spéciaux)

# Guide d'utilisation du Planning ZeShoes & LGL

Application de planning mensuel pour organiser les interventions ZeShoes et LGL (La Poste).

Todo list d'utilisation

  • Ouvrir le planning : Double-cliquez sur index.html ou faites-le glisser dans votre navigateur (Chrome/Firefox/Edge). Le planning s'ouvre sur le mois courant avec vos données précédentes (sauvegardées automatiquement).
  • Naviguer dans le calendrier :
    • Flèches ← → pour mois précédent/suivant.
    • Bouton central (cible) pour revenir à aujourd'hui.
    • Cliquer sur le titre "Mois Année" → sélecteur manuel (choisir mois/année → Appliquer).
    • Sur mobile/tablette : balayage (swipe) gauche/droite.
  • Ajouter/éditer un événement (nom ZeShoes ou LGL) :
    • Cliquer sur une date du mois.
    • Dans le modal : saisir le nom dans "ZeShoes" et/ou "LGL".
    • Cliquer Valider → nom apparaît coloré dans la cellule.
    • Édition : re-cliquer sur la date pour modifier.
  • Supprimer un événement :
    • Ouvrir modal d'une date avec événement.
    • Cliquer Supprimer → événements effacés.
  • Imprimer le planning :
    • Naviguez au mois désiré.
    • Cliquer icône imprimante (Ctrl+P) → planning imprimable (sans jours gris des mois adjacents).
  • Astuces avancées :
    • Sauvegarde auto : Données persistantes dans le navigateur (localStorage), même après fermeture.
    • Raccourci clavier : Échap pour fermer modal/sélecteur.
    • Support mobile : Entièrement tactile/responsive.
    • Notification verte : Confirme sauvegarde (haut de page, 2s).

Fonctionnalités clés

  • Persistance données (ne perdez jamais un planning).
  • Interface bilingue français (jours/mois).
  • Logos officiels ZeShoes/LGL/La Poste.
  • Impression professionnelle.
